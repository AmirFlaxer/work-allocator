# מדריך התחלה מהירה - שיבוץ שבועי חכם

## מה זה?

כלי לחלוקת עבודה שבועית לצוותים: מי עובד, באיזו עמדה, באיזה יום.

---

## 4 צעדים להתחלה

### 1. הגדר עמדות
בלשונית **עמדות והגדרות**: כתבו שם עמדה, בחרו כמה עובדים נדרשים בה בו-זמנית, ולחצו **הוסף עמדה**.
- עמדה = מקום עבודה / תחנה / תפקיד חוזר
- באותה לשונית בוחרים גם את **ימי העבודה** של השבוע (ברירת מחדל: ראשון עד חמישי)

### 2. הוסף עובדים
בלשונית **עובדים**: לחצו **הוסף עובד**.
- שם, מינימום ומקסימום משמרות לשבוע, מספר שיבוצים ביום, ואילו עמדות מתאימות לעובד
- אפשר גם **ייבוא Excel** של רשימת עובדים קיימת

### 3. צור שיבוץ
בלשונית **שיבוץ**: לחצו **צור שיבוץ**.
- המערכת מחלקת עובדים לעמדות לפי הזמינות, המגבלות והפיזור ההוגן
- עריכה ידנית: לחיצה על תא לבחירת עובד, או גרירת תא על תא להחלפה

### 4. שמור וייצא
- **שמור** - הוספת השיבוץ לארכיון (וממנו נבנים הדוחות החודשיים)
- **PNG** - תמונה להדפסה או שיתוף בוואטסאפ
- **Excel** - קובץ xlsx לעריכה חיצונית

## טיפים מהירים

| פעולה | איך |
|--------|-----|
| ביטול שינוי | כפתורי בטל / בצע מחדש בלשונית שיבוץ |
| נעילת תא | אייקון המנעול בתא - תא נעול לא ישתנה בשיבוץ אוטומטי |
| חופשות ובקשות | לשונית "העדפות": ימים לא זמינים ובקשות שיבוץ לשבוע ספציפי |
| שכפול שבוע | כפתור "שכפל שבוע" מעתיק את השיבוץ מהשבוע הקודם |
| תבניות | "שמור תבנית" ו"טען תבנית" לשיבוצים שחוזרים על עצמם |
| מעבר שבוע | כפתורי קודם / הבא ליד התאריך, ו"שבוע נוכחי" לחזרה |
| מצב לילה | כפתור השמש / ירח בסרגל העליון |
| דוחות | לשונית "דוחות": דוח חודשי, דוח עמדות וגרף היסטורי |

## נגישות ואבטחה

- **נגישות**: המערכת עומדת בתקן WCAG AA - ניגודיות צבעים (כולל צבעי העובדים בטבלה), ניווט מלא במקלדת, וכיבוד העדפת צמצום תנועה.
- **אבטחה**: כל ארגון מקבל סביבת נתונים מבודדת. ההפרדה נאכפת בשרת (Row Level Security), כך שאין גישה למידע של ארגון אחר.

פירוט בכפתור **"אודות"** בסרגל העליון.
